Sistema de Contas a Pagar

Funcionalidades do Sistema:

  • Cadastro de contas a pagar
  • Registro de pagamentos
  • Controle por datas de vencimento
  • Classificação por tipo de título
  • Associação com clientes, fornecedores e funcionários
  • Relatórios e filtros por status
Acessar API REST
Banco de Dados:

O sistema utiliza um banco de dados MySQL com a estrutura conforme diagrama fornecido:

  • Schema financeiro: tbContasPagar, tbTipoTitulo
  • Schema cadastro: tbPessoas
  • Schema seguranca: tbUsuarios
  • Schema dominio: PessoaTipo
Sistema desenvolvido em Java Spring Boot
